# Linktree de Kaikyo


Un **hub centralisé** pour retrouver tous mes réseaux sociaux et suivre mes abonnés en temps réel.  
Projet réalisé pour **mieux organiser ma présence en ligne**, facile à maintenir et à mettre à jour.


## Fonctionnalités

### Abonnés en temps réel
- Affiche le nombre d’abonnés sur Twitch, YouTube VOD, YouTube Rediff, TikTok et Instagram  
- Calcul automatique du total d’abonnés  
- Mise à jour facile des valeurs via JavaScript  

### Liens vers les réseaux sociaux
- Accès rapide à tous mes profils : Twitch, YouTube, TikTok, Instagram  
- Boutons interactifs avec effets de survol et clic  
- Organisation en bulles claires pour une navigation simple  

### Design responsive
- Adapté aux écrans grands et petits  
- Effets d’ombre et de survol pour dynamiser l’interface  
- Image de profil centrée avec bordures et survol stylé  


## Technologies utilisées

- HTML
- CSS
- JavaScript  


## Ce que j’ai appris

- Manipuler le DOM pour afficher des données dynamiques  
- Structurer un projet web simple mais complet  
- Créer un design responsive et esthétique  
- Organiser le code pour qu’il soit facilement maintenable  


## Installation

- Ouvrir `index.html` dans un navigateur  
- Les valeurs des abonnés sont déjà définies dans le fichier JS, tu peux les modifier directement  


## Notes

- Les images doivent rester dans le dossier `image/` pour que le HTML fonctionne  
- Le projet est volontairement simple et modulable pour **ajouter facilement de nouveaux réseaux sociaux**
